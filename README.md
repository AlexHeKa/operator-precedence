# 运算符优先级

## 基本规则
1. 一元运算符优于二元运算符
2. 乘除加减优于大部分运算符
3. 位运算优于逻辑运算

|  乘除加减  | *, /, +, - |
|  位移  | >>, << |
|  判断与比较  | >, >=, <, <=, ==,  |
|  位运算  | &, ^, \|, &&, \|\| |
|  逻辑运算  | &&, \|\| |
